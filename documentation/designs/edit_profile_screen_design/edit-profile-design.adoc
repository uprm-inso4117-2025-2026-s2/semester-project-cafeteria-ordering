Design Description:

- Includes both light and dark mode designs.
- Maintains consistency with the Profile Screen layout and design system.
- Header contains a back navigation arrow and the title "Edit Profile".
- Profile picture section includes an edit icon to indicate update functionality.
- "About you" section organizes user information clearly.
- Editable fields:
  * Full Name (required)
  * Username
  * Email (required)
  * Phone Number
  * Password (with "Change" action)
- Required fields are marked with an asterisk (*).
- Clear separation of actions:
  * "Save" (primary action)
  * "Discard Changes" (secondary action)
- Layout is mobile-first, clean, and easy to navigate.
- Input fields are visually distinguishable as editable.
- Designed to support validation, error states, and success feedback.

--

== Key Constraints:
- Must follow existing Profile Screen design.
- Maintain UI design system consistency.
- Inputs must clearly indicate editable fields.
- Must include validation and feedback states.
- Responsive design for mobile and desktop.
- Accessibility considerations required.
