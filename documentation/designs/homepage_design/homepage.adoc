== Overview

This document provides an overview of the final homepage design for the Cafeteria Ordering System. The design was created in Figma based on the approved wireframe and follows the official color palette and typography guidelines defined in the project documentation. The homepage design follows the layout defined in the wireframe from:

- Task #20
- Pull Request #91

== Main Components

The homepage layout includes:

- Header with logo, greeting, and cart icon
- Search bar for finding menu items
- Featured "Best Seller" section
- Category filters (Rating, Breakfast, Lunch)
- Menu item cards displaying food information
- Bottom navigation bar (Home, Orders, Profile)
