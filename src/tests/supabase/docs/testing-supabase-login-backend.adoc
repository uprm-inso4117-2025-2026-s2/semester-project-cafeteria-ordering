= Testing Supabase Login Backend
:toc:
:icons: font

This test validates backend login behavior independently from the frontend login screen.

== Quick Run

[source,bash]
----
npm run test:login-backend
----

This script validates:

* Email/password sign-in
* Session creation after login
* Sign-out behavior
* Session cleanup after logout

== Prerequisites

. Ensure `.env` has Supabase credentials and test account credentials:
+
[source,bash]
----
EXPO_PUBLIC_SUPABASE_URL=https://your-project-ref.supabase.co
EXPO_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here
SUPABASE_TEST_LOGIN_EMAIL=test-user@example.com
SUPABASE_TEST_LOGIN_PASSWORD=your-test-password
----

. Ensure the test account is confirmed in Supabase Auth.
