== Description
All unit tests executed for order modification logic passed successfully. The test suite validated:

- Base meal initialization returns the expected base price and ingredient list.
- Adding a valid extra (`cheese`) increases the base price by the extra's price and appends the ingredient.
- Removing an ingredient removes it from the list and preserves price correctly when the removed ingredient has zero price.
- Duplicate ingredient additions are prevented and do not change the price.
- Direct modification of base price using `modBasePrice()` updates the price correctly.
- Direct replacement of ingredient lists using `modIngredients()` updates the ingredient list without modifying base price.

== Notes
- To prevent test interference, `createBaseMenuItem()` now clones the base ingredient array for each test instance so modifications do not persist across tests.
